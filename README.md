# 🌍 GlobalTranslate AI

A Telegram bot that translates text into multiple languages using LibreTranslate.

## Features

- Automatic language detection
- Translate to multiple languages
- Telegram inline buttons
- Railway Ready
- GitHub Ready

## Deploy

1. Fork or upload to GitHub
2. Connect GitHub to Railway
3. Add:

BOT_TOKEN

TRANSLATE_API

4. Deploy

Made with Python + aiogram
